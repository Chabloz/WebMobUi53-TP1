# IM53 TP1

This project uses Vue 3 with Vite.

## Requirements

- Node.js
- npm

## Install

```bash
npm install
```

## Run in Development

```bash
npm run dev
```

Vite will print the local development URL in the terminal, usually `http://localhost:5173/`.

## Other Useful Commands

```bash
npm run build
npm run preview
```
